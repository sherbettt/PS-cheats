## 1. Просмотр пользователей

### Локальные пользователи:
```powershell
# Все локальные пользователи
Get-LocalUser

# С фильтрацией
Get-LocalUser | Where-Object {$_.Enabled -eq $true}  # Только активные
Get-LocalUser | Select-Object Name, Enabled, Description, LastLogon

# Конкретный пользователь
Get-LocalUser -Name "ИмяПользователя"
```

### Доменные пользователи (если есть домен):
```powershell
# Требуется модуль ActiveDirectory
Import-Module ActiveDirectory

# Все пользователи домена
Get-ADUser -Filter * | Select-Object Name, SamAccountName, Enabled

# Поиск конкретного
Get-ADUser -Identity "username"
```

## 2. Управление группами

### Просмотр групп:
```powershell
# Все локальные группы
Get-LocalGroup

# Члены конкретной группы
Get-LocalGroupMember -Group "Administrators"
Get-LocalGroupMember -Group "Пользователи"

# Только имена членов
(Get-LocalGroupMember -Group "Administrators").Name
```

### Добавление пользователей в группы:
```powershell
# Добавить одного пользователя в группу администраторов
Add-LocalGroupMember -Group "Administrators" -Member "UserName"

# Добавить в другую группу (например, "Пользователи удаленного рабочего стола")
Add-LocalGroupMember -Group "Remote Desktop Users" -Member "UserName"

# Добавить нескольких пользователей
$users = @("User1", "User2", "User3")
foreach ($user in $users) {
    Add-LocalGroupMember -Group "Administrators" -Member $user
}

# Добавить пользователя с проверкой существования
$user = "UserName"
$group = "Administrators"

if (Get-LocalUser -Name $user -ErrorAction SilentlyContinue) {
    Add-LocalGroupMember -Group $group -Member $user
    Write-Host "Пользователь $user добавлен в группу $group" -ForegroundColor Green
}
```

### Удаление из групп:
```powershell
# Удалить из группы администраторов
Remove-LocalGroupMember -Group "Administrators" -Member "UserName"

# Удалить из группы с подтверждением
Remove-LocalGroupMember -Group "Administrators" -Member "UserName" -Confirm
```

## 3. Управление пользователями

### Создание пользователя:
```powershell
# Простое создание
New-LocalUser -Name "NewUser" -Password (Read-Host -AsSecureString "Введите пароль") -FullName "Полное Имя" -Description "Описание"

# Создание с параметрами
$Password = Read-Host -AsSecureString "Введите пароль"
New-LocalUser -Name "NewUser" `
              -Password $Password `
              -FullName "Иван Иванов" `
              -Description "Сотрудник отдела IT" `
              -AccountNeverExpires `
              -UserMayNotChangePassword

# Создание пользователя и добавление в группу
$user = New-LocalUser -Name "NewUser" -Password $Password -AccountNeverExpires
Add-LocalGroupMember -Group "Users" -Member "NewUser"
```

### Удаление пользователя:
```powershell
# Удалить пользователя
Remove-LocalUser -Name "UserName"

# Удалить с подтверждением
Remove-LocalUser -Name "UserName" -Confirm

# Удалить пользователя и проверить его наличие
$user = "UserName"
if (Get-LocalUser -Name $user -ErrorAction SilentlyContinue) {
    Remove-LocalUser -Name $user
    Write-Host "Пользователь $user удален" -ForegroundColor Green
} else {
    Write-Host "Пользователь $user не найден" -ForegroundColor Red
}
```

### Изменение параметров пользователя:
```powershell
# Включить/отключить учетную запись
Disable-LocalUser -Name "UserName"  # Отключить
Enable-LocalUser -Name "UserName"   # Включить

# Сменить пароль
$NewPassword = Read-Host -AsSecureString "Новый пароль"
Set-LocalUser -Name "UserName" -Password $NewPassword

# Изменить свойства
Set-LocalUser -Name "UserName" -Description "Новое описание" -AccountNeverExpires

# Разрешить/запретить смену пароля
Set-LocalUser -Name "UserName" -UserMayNotChangePassword $true
Set-LocalUser -Name "UserName" -UserMayNotChangePassword $false
```

## 4. Настройка привилегий

### Основные группы и их права:

| Группа | Права |
|--------|-------|
| **Administrators** | Полный контроль над системой |
| **Users** | Базовые права, ограниченный доступ |
| **Power Users** | (устарело) Расширенные права пользователя |
| **Remote Desktop Users** | Доступ по RDP |
| **Backup Operators** | Права на резервное копирование |

### Назначение прав через группы:
```powershell
# Основные группы для добавления
$groups = @(
    "Administrators",        # Полный доступ
    "Remote Desktop Users",  # RDP доступ
    "Performance Log Users", # Мониторинг
    "Hyper-V Administrators" # Управление Hyper-V (если есть)
)

# Добавить пользователя в несколько групп
$user = "UserName"
foreach ($group in $groups) {
    try {
        Add-LocalGroupMember -Group $group -Member $user -ErrorAction Stop
        Write-Host "Добавлен в $group" -ForegroundColor Green
    } catch {
        Write-Host "Не удалось добавить в $group: $_" -ForegroundColor Red
    }
}
```

### Настройка специфических привилегий (через secedit или ntrights):

## 5. Полный скрипт для управления пользователем

```powershell
# Скрипт для создания пользователя с полными настройками
function New-AdvancedUser {
    param(
        [Parameter(Mandatory=$true)]
        [string]$UserName,
        
        [Parameter(Mandatory=$true)]
        [string]$FullName,
        
        [string]$Description = "",
        
        [switch]$IsAdmin,
        
        [switch]$AllowRDP
    )
    
    # Запрашиваем пароль
    $Password = Read-Host "Введите пароль для $UserName" -AsSecureString
    
    # Создаем пользователя
    try {
        $user = New-LocalUser -Name $UserName `
                              -Password $Password `
                              -FullName $FullName `
                              -Description $Description `
                              -AccountNeverExpires `
                              -ErrorAction Stop
        
        Write-Host "Пользователь $UserName создан" -ForegroundColor Green
        
        # Добавляем в базовую группу Users
        Add-LocalGroupMember -Group "Users" -Member $UserName
        Write-Host "Добавлен в группу Users" -ForegroundColor Gray
        
        # Добавляем в администраторы, если нужно
        if ($IsAdmin) {
            Add-LocalGroupMember -Group "Administrators" -Member $UserName
            Write-Host "Добавлен в группу Administrators" -ForegroundColor Yellow
        }
        
        # Добавляем в RDP, если нужно
        if ($AllowRDP) {
            Add-LocalGroupMember -Group "Remote Desktop Users" -Member $UserName
            Write-Host "Добавлен в группу Remote Desktop Users" -ForegroundColor Gray
        }
        
        # Показываем информацию
        Get-LocalUser -Name $UserName
        Get-LocalGroupMember -Group "Administrators" | Where-Object {$_.Name -like "*$UserName*"}
        
    } catch {
        Write-Host "Ошибка: $_" -ForegroundColor Red
    }
}

# Использование:
# New-AdvancedUser -UserName "Ivanov" -FullName "Иван Иванов" -IsAdmin -AllowRDP
```

## 6. Скрипт для удаления пользователя

```powershell
function Remove-UserCompletely {
    param(
        [Parameter(Mandatory=$true)]
        [string]$UserName
    )
    
    # Проверяем существование
    if (Get-LocalUser -Name $UserName -ErrorAction SilentlyContinue) {
        
        # Удаляем из всех групп (можно не делать, т.к. пользователь будет удален)
        $groups = Get-LocalGroup
        foreach ($group in $groups) {
            $members = Get-LocalGroupMember -Group $group.Name -ErrorAction SilentlyContinue
            if ($members.Name -like "*$UserName*") {
                Remove-LocalGroupMember -Group $group.Name -Member $UserName
                Write-Host "Удален из группы $($group.Name)" -ForegroundColor Gray
            }
        }
        
        # Удаляем пользователя
        Remove-LocalUser -Name $UserName
        Write-Host "Пользователь $UserName удален" -ForegroundColor Green
        
    } else {
        Write-Host "Пользователь $UserName не найден" -ForegroundColor Red
    }
}

# Использование:
# Remove-UserCompletely -UserName "Ivanov"
```

## Важные замечания:

1. **Все команды требуют запуска PowerShell от имени администратора**

2. **Для доменных пользователей** используйте `Get-ADUser`, `Add-ADGroupMember` и т.д.

3. **Безопасность**: 
   - Не давайте права администратора без необходимости
   - Используйте сложные пароли
   - Регулярно проверяйте список администраторов

4. **Проверка текущих прав**:
```powershell
# Кто сейчас администратор
Get-LocalGroupMember -Group "Administrators"

# Какие группы у пользователя
$user = "UserName"
Get-LocalGroup | Where-Object {
    (Get-LocalGroupMember -Group $_.Name).Name -like "*$user*"
} | Select-Object Name
```
